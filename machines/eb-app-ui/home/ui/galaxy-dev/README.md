#### clean up

```bash
cd /home/ui/galaxy-dev

rm package-lock.json
rm yarn.json
rm -rf node_modules
rm -rf .svelte-kit

yarn install
```

#### check

```bash
yarn run check
yarn run lint
deno fmt --check src
```

```bash
(cd /home/ui/galaxy-dev && yarn run dev --host --port 3000)
```
